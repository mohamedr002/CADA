## MNIST Baseline    
In this readme, give instructions on how to run your code.   

#### CPU   
```bash   
python mnist_baseline_trainer.py     
```

#### Multiple-GPUs   
```bash   
python mnist_baseline_trainer.py --gpus '0,1,2,3'  
```   

#### On multiple nodes   
```bash  
python mnist_baseline_trainer.py --gpus '0,1,2,3' --nodes 4  
```
